没有服务器做一个网站(1)

发表于 2023-08-03

没有服务器做一个网站(1)

你是否已经有了自己的网站了？
如果有了请跳过，
如果没有自己的网站，本文章会讲没有服务器怎么做一个网站。

内网穿透

确保你已经安装了Node.js和npm，并在命令行中可用。
打开命令行终端，并进入你的项目目录。
在项目目录中初始化一个新的npm项目，执行以下命令：
npm init
按照提示填写项目信息并完成初始化。
安装localtunnel依赖，执行以下命令：
npm install -g localtunnel
启动你的本地服务器（例如使用Express框架）。
在命令行中执行以下命令来启动localtunnel：
lt --port <your_local_server_port>
是你本地服务器运行的端口号，例如3000。
localtunnel会为你分配一个公共的URL，它将转发到你的本地服务器。你可以在命令行窗口中看到分配的URL。
在浏览器中访问分配的URL，即可访问你的本地站点。

利用github搭建网页游戏

发表于 2023-08-03

利用github搭建游戏网站

第一步注册账号

  1. 打开https://github.com/
  2. 注册账号

获取源码

我们以Chrome 的离线小恐龙为例

  1. 打开https://github.com/wayou/t-rex-runner
  2. 下载源码

部署网站

  1. 点击github右上角的加号
  2. 创建一个存储库
  3. 上传源码到存储库
  4. 点击设置
  5. 点击Pages
  6. 点击Branch下面的下拉项
  7. 点击main
  8. 点击右边的Save
  9. 等待一会，你就能看到自己的网址了
