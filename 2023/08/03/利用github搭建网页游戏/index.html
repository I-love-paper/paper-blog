利用github搭建游戏网站

第一步注册账号

  1. 打开https://github.com/
  2. 注册账号

获取源码

我们以Chrome 的离线小恐龙为例

  1. 打开https://github.com/wayou/t-rex-runner
  2. 下载源码

部署网站

  1. 点击github右上角的加号
  2. 创建一个存储库
  3. 上传源码到存储库
  4. 点击设置
  5. 点击Pages
  6. 点击Branch下面的下拉项
  7. 点击main
  8. 点击右边的Save
  9. 等待一会，你就能看到自己的网址了
